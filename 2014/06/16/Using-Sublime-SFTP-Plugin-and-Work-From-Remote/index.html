Using Sublime SFTP Plugin and Work From Remote

[Sublime] is my favorite editor. Sublime does have Linux verison, but it is not compatible to RHEL5.6 on my work desktop. I miss Ubuntu, however, RHEL5.6 is required by the company where I have my intern. Fortunetely, I find a alternate way to use sublime. Instead of coding directly on my work desktop, I can edit source code on my laptop and sync them to the desktop. The solution is Sublime SFTP.

Sublime SFTP is a plugin for Sublime which help us sync our local code and remote code.

1. Installation of Sublime SFTP

  1. The Sublime SFTP could be installed by Package Control. Install it first if you have not done so.
  2. shift + ctrl + p, type in “pci” to start Package Control.
  3. Type in “SFTP” and hit enter to install SFTP automatically.

2. Setup server config

After Installation, we need to specify the server config. Click File menu and Click SFTP/FTP (It could be also started by shift + ctrl + p, and type in “SFTP”), select Setup Server. A new file will pop up. We can leave most of settings as default. Please set your config to following values:
